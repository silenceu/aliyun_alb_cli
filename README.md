```
USAGE
  $ xj0_alb_cli hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ xj0_alb_cli hello
  hello world from ./src/hello.ts!
```
